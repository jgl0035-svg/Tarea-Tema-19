En esta actividad hemos trabajado tres compañeros: José Miguel, José Luis e Ignacio. Cada uno creó su propia rama y modificó los mismos archivos para generar un conflicto real. Al integrar los cambios, Git detectó el conflicto y yo me encargué de resolverlo manualmente, dejando los archivos corregidos y la rama main limpia.



Para cumplir las funcionalidades avanzadas, utilizamos git stash para guardar un cambio temporal y luego recuperarlo con git stash pop. También realizamos una mejora del historial usando git rebase -i, donde reescribimos el mensaje de un commit.
